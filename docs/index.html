Quickstart

Up and running in 60 seconds

01

Install via npm or run with npx

No global install required. Run directly with npx or install once and use everywhere. Zero production dependencies.

npx readmeai init
02

ReadMeAI scans your project

It reads your package.json, detects your framework, linting config, folder structure, and git history to build deep context.

readmeai analyze --depth deep
03

Context files are generated instantly

CLAUDE.md, .cursorrules, copilot-instructions.md — all written to your repo. Commit them and every AI tool on your team gains instant project knowledge.

git add CLAUDE.md .cursorrules && git commit -m "feat: add AI context"
